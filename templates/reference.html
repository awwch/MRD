О проекте

Мы занимаемся созданием первого машиночитаемого словаря для русского языка на базе нескольких толковых словарей, выполненного на основе стандарта TEI.

Конечным результатом нашей работы будет ресурс, позволяющий осуществлять поиск по словарным статьям и их элементам. Таким образом, продукт будет полезен для широкого круга пользователей, в том числе для лингвистов, филологов, а также для людей, занимающихся изучением русского языка.

Мы предполагаем возможность дополнения словаря информацией из других лексикографических источников пользователями, для чего размещаем исходный код в свободном доступе здесь.
